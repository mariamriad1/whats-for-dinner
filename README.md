# 🍽️ What's For Dinner?

A simple recipe website that displays a random recipe each time the user clicks the **"Try Another Recipe"** button.

## Live Demo

🔗 https://mariamriad1.github.io/whats-for-dinner/

## Features

- Random recipe generator
- Recipe image
- Ingredients
- Instructions
- Nutrition facts
- Chef's tips
- Responsive design

## Built With

- HTML
- CSS
- Bootstrap
- JavaScript
